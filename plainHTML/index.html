Current Research

  • Mobile Quality of Experience
    In todayâ€™s world, smartphones have become predominant in Internet usage. It is important to understand and improve the quality of experience for mobile Internet applications. Typically, I focus on the applications such as video streaming (YouTube), telephony (Skype) and Web browsing etc. The problem here is that, a number of factors (such as network, hardware, inherent application complexity etc) influence the quality of experience. Therefore, pinpointing and diagnosing the problem, has become a challenging task for these mobile Internet applications. We try to solve these problems (mostly using machine learning) and improve the system experience.

  • Next Generation Cellular Networks
    Cellular traffic is going to be very diverse in near future. This is because of the cellular enabled IoT, wireless spectrum co-existence and prolific increase in smartphone usage. The current radio access network and packet core network are not well suited for this traffic, especially for the control plane. For example, the core components such as MME, PGW, SGW and HSS are tighly coupled together with users' states, making the packet core vulnerable to failure and overloads. We bring software defined changes such as network function virtualization and improve the scalability of next generation cellular experience.
